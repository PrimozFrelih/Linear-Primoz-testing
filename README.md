# Simple Todo App

A minimal single-page to-do app built with plain HTML, CSS, and JavaScript.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks complete or incomplete
- Persist tasks in browser `localStorage`

## Tech

- HTML
- CSS
- JavaScript
- No backend
- No external dependencies

## Project structure

- `index.html` — app structure
- `style.css` — styles and layout
- `script.js` — app logic and persistence

## Behavior

- Tasks are created from the input field
- Empty tasks are rejected
- Tasks can be edited and deleted
- Completion state can be toggled
- Tasks persist after page refresh using `localStorage`

## Current scope

This project is intentionally simple and local-only.

Out of scope:
- authentication
- backend/database
- multi-user sync
- notifications
- categories or filters
- due dates

## Run locally

Open `index.html` in a browser.
